# 紫微斗數學習網站 — 專案計畫

## 網站目標

建立一個結構清晰、由淺入深的紫微斗數線上學習平台，讓完全沒有基礎的初學者能夠系統性地理解紫微斗數的核心概念與解盤邏輯。

## 使用者輪廓

- 繁體中文使用者
- 沒有紫微斗數基礎
- 偏好有邏輯、有結構的學習方式
- 希望理解「為什麼」而非只背誦「是什麼」
- 可能對命理有興趣但不希望被迷信語調影響

## 內容架構

### 學習路線

1. **認識紫微斗數** — 是什麼、為什麼學、怎麼學
2. **基礎觀念** — 命盤結構、十二宮、星曜分類、四化
3. **十二宮詳解** — 逐宮深入理解
4. **十四主星** — 逐星拆解
5. **輔星與煞星** — 輔助判讀
6. **四化詳解** — 化祿、化權、化科、化忌
7. **解盤教學** — 步驟化引導
8. **練習案例** — 虛構範例實作
9. **詞彙表與常見問答**

## 技術堆疊

- **框架**: Next.js 16 (App Router)
- **語言**: TypeScript
- **樣式**: Tailwind CSS v4
- **圖示**: Lucide React
- **部署**: 靜態輸出，可部署至任何靜態主機
